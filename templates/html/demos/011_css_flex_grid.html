Flex 与 grid 的区别， 主要的应用场景。
https://css-tricks.com/quick-whats-the-difference-between-flexbox-and-grid/
Grid 也可以实现 flex 的多数功能，用 fr 设置比例， px 设置固定值， 而且 grid 的布局一般更确定。
所有的 flex 外容器共用一个 class="flex-box", 子元素用 class="item"
排列有 row column, 默认是row, 默认不会换行
flex-wrap 设置 wrap, 也可以 flex-flow 一起设置两个属性。
justify-content 设置 main axis 剩余空间如何分配。
align-content 设置在 cross axis 剩余空间如何分配
flex-grow 的属性规定为一个 number, 负值无效，默认为 0。
这个属性规定了 flex-grow 项在 flex 容器中分配剩余空间的相对比例
flex-basis 指定了 flex 元素在主轴方向上的初始大小。
父元素只指定了 flex, 布局由子元素的 flex 属性来控制。

fr (fractional unit): 1fr 就表示一份的可用空间，相当于把所有 fr 表示的空间按比例分配。
Grid 布局可以行高和列宽，并给每个格子指定名字，然后可以把子元素按名字指定格子。
